方案：滚动条作为编辑器的内联子元素

在 Editor::render() 中，在 line_list 旁边渲染一个滚动条 div。结构上简单直接：

div()
  .size_full()
  .child(line_list)       // ← 已有的列表
  .child(scrollbar())     // ← 新增：右侧浮动的滚动条

数据流

总内容高度 = 已测量行高度之和 + (未测量行数 × 默认行高)
滑块大小 = (视口高度 / 总内容高度) × 轨道高度
滑块位置 = (当前滚动位置 / 总内容高度) × 轨道高度
拖拽 → scroll_by(delta)
